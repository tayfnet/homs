# TAIF Public Price Board

مجلد مستقل لشاشة الأسعار العامة فقط.

## التشغيل

افتح:

```text
public-price-board/index.html
```

أو ارفع مجلد `public-price-board` كاملاً إلى دومين مستقل.

## العزل

- لا يحمل واجهة المشروع الأساسية.
- لا يحتاج تسجيل دخول.
- يقرأ أسعار العملات فقط.
- لا يكتب أو يعدل في قاعدة البيانات.
- التصميم داخل `css/price-board.css` مأخوذ من نفس ملفات شاشة الأسعار الداخلية في هذه النسخة.
- أعلام العملات داخل المجلد حتى يعمل على دومين منفصل.

## الخطوط

ملفات الخط الثنائية غير مرفقة داخل مجلد الشاشة. إذا أردت نفس الخط المحلي تماماً عند رفع المجلد وحده، انسخ مجلد الخطوط من نسختك الأصلية إلى:

```text
public-price-board/assets/fonts/noto-naskh-arabic/
```
